# Background Music

Place your copyright-safe YouTube Audio Library MP3 here:

```text
background-music.mp3
```

Recommended search style inside YouTube Studio Audio Library:

- Genre: Dance & Electronic or Pop
- Mood: Bright, Happy, or Funky
- Attribution: Attribution not required
- Length: 1-3 minutes

The game automatically uses this file for background music. If the file is missing, it falls back to the built-in generated melody.
